Nails inspo: App de unhas criativas

O Nails inspo é um app gratuito onde qualquer pessoa pode navegar por um catálogo de unhas de estilos variados. O projeto foi feito com HTML para estrutura, CSS para o visual e JavaScript para interatividade.

A ideia é criar uma plataforma leve e acessível para usuarios acharem ideais para novos estilos de unha.
